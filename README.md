# To-Do Application 📝

Welcome to the **To-Do Application**, a platform designed to help users manage their tasks efficiently while collaborating in groups. This app provides seamless task management, user authentication, and group collaboration tools, making productivity effortless!

---

## 🌟 Features

- **User Authentication** 🔐
  - Sign up and log in using email and password.
  - Google OAuth for easy and secure login.

- **Task Management** ✅
  - Create, assign, and track tasks.
  - Task statuses: Pending, In Progress, and Completed.

- **Group Collaboration** 👥
  - Create groups and invite team members.
  - Assign tasks to specific group members.

- **User Roles** 🎭
  - Group members can be assigned roles like **Admin** or **User**.
  - Admins manage tasks and users within the group.

---

## 🖥️ How It Works

1. **Sign Up / Log In**:  
   Users can register using their email or log in through Google OAuth for a seamless experience.

2. **Create Groups**:  
   Organize tasks by creating groups for different projects.  

3. **Assign Tasks**:  
   Assign tasks to team members, set due dates, and track progress.

4. **Track Progress**:  
   Monitor task statuses and group activity in real-time.

---

## 📊 User Roles and Functions

| Role        | Functionality                                   |
|-------------|------------------------------------------------|
| **Admin**   | Create groups, manage members, and assign tasks. |
| **User**    | View tasks, update statuses, and collaborate.   |

## 🛠️ Tech Stack

- **Authentication**: Passport.js, Google OAuth
